# Caesar Cipher

Uses Python's nltk to encrypt, decrypt and crack messages

- encrypt():
    - arguments: phrase, key
    - return: encrypted message

- decrypt():
    - arguments: phrase, key
    - return: decrypted message

- crack():
    - arguments: phrase
    - return: decrypted message
